ImageAnalysis Module (`sbpy.imageanalysis`)
===========================================

Introduction
------------

`~sbpy.imageanalysis` provides routines for the enhancement of
structure in comet images (`~sbpy.imageanalysis.CometEnhancement`) and
the subtraction of PSF for imaging data to search for extended
emission (`~sbpy.imageanalysis.PSFSubtraction`).
